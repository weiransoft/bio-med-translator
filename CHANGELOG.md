# 变更日志

本文档记录 bio-med-translator Skill 的所有重要变更。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [5.1.0] - 2026-03-06

### Fixed

- 页眉页脚中文翻译问题
  - 添加页眉页脚翻译处理逻辑
  - 翻译页眉中的文档标题、文件编号、版本号、页码
  - 翻译页脚中的公司名称和其他中文内容
  - 确保文档完全翻译为英文，无中文残留

### Added

- 页眉页脚翻译要求检查清单
  - 文件编号 → Document No.
  - 版本号 → Rev No.
  - 页码 → Page
  - 第 X 页 共 X 页 → Page X of X

## [5.0.0] - 2026-03-06

### Changed

- Word 原生格式保持翻译功能
  - 使用 Python 脚本和 python-docx 库实现
  - 完整保持原文档格式（字体、段落、样式、表格、页面布局）
  - 输出仍为 Word 文档 (.docx)

- 文件读取方式更新
  - 使用 Python 脚本替代 Trae Read 工具读取 Word 文档
  - 使用 python-docx 库解析文档内容和格式信息

- 文档结构简化
  - 移除详细实现代码
  - SKILL.md 简化为技能定义文件
  - 详细使用说明移至 README.md

### Removed

- SKILL.md 中的 Python 代码示例
- 详细实现步骤中的代码块

## [4.2.0] - 2025-01-13

### Changed

- 翻译核心原则强化
  - 严格忠实原文，禁止臆想和添油加醋
  - 明确翻译边界和允许/禁止操作
  - 添加特殊情况处理指南

### Added

- 术语一致性检查清单
- 格式完整性检查清单

## [4.1.0] - 2025-01-13

### Changed

- 自动文件名翻译功能
- GitHub 提交功能集成

## [4.0.0] - 2025-01-13

### Changed

- 支持多种文件格式翻译
  - Word (.docx, .doc)
  - PDF (.pdf)
  - Markdown (.md)
  - Excel (.xlsx, .xls)

- 术语库系统
  - 核心术语库
  - 领域特定术语库（抗体药物、基因组学、小核酸制药、临床医学）
  - 术语管理脚本

- 质量控制
  - 多轮审校流程
  - 术语一致性检查
  - 格式完整性验证

## [3.0.0] - 2024-12-20

### Changed

- 初始版本
- 支持中英互译
- 文档类型识别
- 领域识别
